Frontend: Interface & Experiência

Tecnologias: Next.js 15, React 19, Tailwind CSS e Chart.js para visualizações ricas e responsivas.
Dashboards Personalizados: Interfaces adaptadas para cada tipo de usuário (aluno, professor, administrador).
Formulários Dinâmicos: Campos específicos por perfil de usuário no registro e formulários adaptados por tipo de operação.
Integração com Backend: Context API para gerenciamento de estado, token JWT para autenticação, upload e visualização de imagens em tempo real.
